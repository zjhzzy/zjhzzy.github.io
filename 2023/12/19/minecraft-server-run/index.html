我的世界（Minecraft）开服指北

JinHong Zeng
2023-12-19
Minecraft

想要和自己的朋友一起玩我的世界，但是朋友不在身边？

有空闲的服务器想要发挥它的作用？

看我这个教程就对了

服务端类型

我的世界有两大类Java和Bedrock，这两个大类下又有几个小类

  • Java
    • 原版端 顾名思义就是Minecraft官方发布的端，啥都不能添加
    • 插件端 这类服务端只能添加插件也被称为纯净端，但是技术够厉害的也能做出和mod差不多的插件，最具代表性的为Spigot/Paper
    • 模组端 这类服务端只能添加模组也被称为模组端，最具代表性的就是Forge/Fabric
    • 插件模组混合系 这类服务端既能添加插件也能添加模组，最具代表性的是Arclight/Mohist
    • 代理端 这个我也不太熟悉就不介绍了
  • Bedrock
    • 原版端 同上
    • Nukkit系 与Java的插件端差不多，最具代表性的是Nukkit/PocketMine服务端
      现在正式开始教程，各个服务端之间都有些许差别，本教程是基于Spigot系而做的教程

开始教程

安装JDK

Java Downloads | Oracle 中国 选择一个适合自己的版本下载

下载服务端

下载服务端到一个空目录下，我这里给出Spigot的下载链接
此时目录应该是这样的（如下图）

服务端启动！！！

启动终端，进入到jar文件在的那个目录，进行第一次启动

执行完这条命令一会后你会发现程序终止了，目录下多出了几个文件，我来介绍一下

eula.txt

这个是有关eula协议的文件，需要将false改成true同意该协议服务端才能正常启动

server.properties

这个是服务端的配置文件，我这里讲解几个基础的配置项

server-port：这个是服务端发送数据的端口，默认为25565
online-mode：这个是服务端是否是在线模式，如果你或你朋友没有正版账号的话就设置为false
gamemode：服务端的模式，默认为生存
motd：服务器在游戏中显示的介绍

再次启动

还是同样的命令，这一次没有意外的话服务器就正常启动了

出现这句话那服务器应该是完成加载了

添加插件

插件端正常启动后一般都会添加一个plugins文件夹，模组端是mods文件夹

如果需要添加插件或模组就将下载来的jar文件拖入到对应的文件夹中，再重启服务端就可以了
输入 plugins 命令就会显示安装了多少个插件，并以颜色区分插件是否正常加载启动

结语

本开发指北未完待续，我暂时想不出有哪些需要写进去的了，除非腐竹是一个电脑小白，本教程对于开一个只是与朋友玩的服务器是足够的了
